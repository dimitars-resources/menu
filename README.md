## Features
* Modern UI
* Easy to install
* Give items with ease
* Create items with ease

## Requirements
* ESX Framework
* mysql-async

## Installation
* Drag the `menu` folder into your `resources` folder.
* Add `ensure menu` in your `server.cfg` making sure it is below `mysql-async`.
* Enjoy!
